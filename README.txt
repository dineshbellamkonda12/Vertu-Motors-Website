#Vertu Motors Website   

- This website is optimized for both mobile and desktop(Full Screen) viewing
- Please open index.html to open the main home page

- Desktop (Full Screen View)
    -Top Navigation Bar
        - The search bar will show results when you type text
        - The Mic button in search bar will listen to your voice
        - The Find Us will take you to another page and show the Vertu Motors map location
        - The Save Icon with Heart Symbol will save the current view of webpage in JPG image format
        - Click on Login Button, Enter username and password and click on Login (This will save username in session storage and will display this username in loggedinform.html)
        - Hover on menus (New Cars, Used Cars, Servicing, Commercials) to open drop downs

    -BMW I4 Trims Section
        - In BMW I4 Trims Section Scroll the Cars to right or left using left and right buttons


- Mobile View
    - The Hamburger Menu on Left will open and show the menus with dropdowns
    - The Search Icon will open the Search Bar
    - The Heart Love Icon will save the current view of the webpage in JPG Image format
    - The Human Icon Next to heart symbol will implement simple login functionality
    - The Phone Icon will initiate a phone call
